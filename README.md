# My Xonsh Setup


## Installation

```shell
mkdir ~/xonsh/
cd ~/xonsh/
git clone https://github.com/siecje/xonsh-setup
cd xonsh-setup/
```

sh
```shell
./install.sh
```

fish
```shell
./install.fish
```

Python3
```shell
./install.py
```

## Upgrade

```shell
upgrade_xonsh
```

### Set as Default shell

```shell
chsh -s `which xonsh`
```
